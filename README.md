# Проект Next.js + FastAPI

Этот проект сочетает фронтенд на **Next.js** и бэкенд на **Django Rest Framework**.  

## Запуск проекта

Проект запускается с помощью **Docker Compose**. Для этого выполните команду:

```bash
sudo docker compose up --build
```

после запуска докера страничка будет доступна по адресу **http://127.0.0.1:3000**
